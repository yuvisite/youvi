Getting started

Youvi is a local video platform that runs entirely in the browser. To get started, you need to select a folder with video files.

First step: Open Management page and click "Select folder" to connect a folder with videos.

Requirements

  • Modern browser with File System Access API support (Chrome, Edge)
  • Local folder with video files (MP4, WebM, MKV)
  • Folder access permission on first launch

Quick start

  1. Open management page
  2. Click "Select folder" and specify folder with videos
  3. Wait for file scanning
  4. Go to main page for viewing

Video metadata format

File .metadata/{video}.meta.json contains tags and statistics:

{
  "views": 42,
  "likes": 5,
  "dislikes": 0,
  "tags": ["anime (ct)", "comedy (ge)", "Channel (ch)"],
  "created": 1701619200000,
  "description": "Video description"
}

IndexedDB (database 8SiteDB)

  • 8Site is an old name of Youvi. (Because site was created at 8 Aug and it was a temporary name)

LocalStorage

  • sidebarCollapsed â€” sidebar state

Important: All data is stored locally. When deleting .youvi/, .channels/ or .metadata/ folders, corresponding data will be lost.
